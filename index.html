How to Use

  • Click buttons or use keyboard to perform calculations
  • AC - Clear all (reset calculator)
  • CE - Clear entry (remove last entry)
  • % - Percentage (divide by 100)
  • = - Calculate result

Keyboard Support: Use numbers (0-9), operators (+, -, *, /), Enter (=), Escape (AC), and Backspace (CE)
